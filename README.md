little game made in C with the mlx graphic library (https://github.com/codam-coding-college/MLX42) that can be played by passing a "map" of a certain format as an argument.

map exemple :  
11111111111  
100001C0011  
1CE00100011  
11101110111  
1000C0P0011  
11111111111
